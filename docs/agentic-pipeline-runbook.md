# Agentic Pipeline Runbook

This file is kept as the legacy entry point for agentic pipeline work. The
detailed contracts now live in focused runbooks:

- [Agentic Pipeline Framework](runbooks/agentic-pipeline.md)
- [Document Ingestion](runbooks/document-ingestion.md)
- [AI Assistant Search](runbooks/ai-assistant-search.md)

Use the framework runbook when changing generic pipeline execution, provider
interfaces, instrumentation, telemetry, prompts, schemas, or live provider
smoke checks.

Use the document ingestion and AI assistant search runbooks when changing
PaperBridge product workflows built on top of the generic pipeline foundation.

For current command coverage, see [Validation](validation.md).
